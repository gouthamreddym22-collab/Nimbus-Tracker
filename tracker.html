Nimbus — Progress

Working title — product name TBD

Nimbus is a continuous personal AI companion — accessible across phone, desktop, and web — that develops a persistent relationship with each user over years. Built on a three-layer architecture: harness substrate (Hermes-absorbed + Claude-Code patterns) / orchestration brain (five verbs + persistent identity + auto-handoff) / regular-user kernel (three canvases + tile patterns + 32 kernel primitives).

Differentiating commitments: full-duplex voice steering, trustless hybrid vault, Device-Service Mesh (universal device + service routing), hot-GPU per-session conversational inference, tile-disambiguation as the consumer pattern for ambiguity, and an auto-Skill-acquisition loop so the agent improves through use.

This tracker measures honest progress toward Phase 1 closed alpha: 50–100 invited users across consumer archetypes, on an 8-vertical kernel (Calendar / Comms / Food / Home Auto / Information / Bills+Payments + desktop Document Editing + Local Files+Code). Numbers are deliberately conservative.
