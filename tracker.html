Nimbus — Progress

Working title — product name TBD

Nimbus is a continuous personal AI companion — accessible across phone, desktop, and web — that develops a persistent relationship with each user over years. Built on a three-layer architecture: harness substrate (Hermes-absorbed + Claude-Code patterns) / orchestration brain (five verbs + persistent identity + auto-handoff) / regular-user kernel (three canvases + tile patterns + 32 kernel primitives).

Differentiating commitments: full-duplex voice steering, trustless hybrid vault, Device-Service Mesh (universal device + service routing), hot-GPU per-session inference, Private Inference Substrate (Premium-tier sealed-enclave for private data), Common Knowledge Substrate (centralized multimodal world-state ingest), tile-disambiguation as the consumer ambiguity pattern, and an auto-Skill-acquisition loop so the agent improves through use.

This tracker measures honest progress toward two product rungs. V1 — single-client demo: phone + PC clients on a local PC server with Claude OAuth; essential 5-verb surface; demoable to one prospect at a time; no partner-compute dependency. Alpha — closed alpha: 50–100 invited users across consumer archetypes; full feature suite on an 8-vertical kernel; partner-compute data centre.
